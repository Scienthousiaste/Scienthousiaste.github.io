Le style avec les Cascading Style Sheet

Syntaxe du CSS

_sélecteur_ {
  propriété1: valeur;
  propriété2: "valeur en plusieurs mots";
}

Le sélecteur peut être :

  • Une balise : p {
  • Un identifiant (attribut id) : #identifiant {
  • Une classe (attribut class) : .classe {
  • Une balise/class dans une autre : p ul .elem{
  • Plusieurs sélecteurs : p, ul .elem{

Display

La propriété display permet d'indiquer la manière dont l'élément se comporte

  • inline : l'élément s'affiche en ligne, les propriétés width et height ne peuvent pas être appliquées
  • block : passe à la ligne et prend toute la largeur
  • inline-block : ne passe pas à la ligne, mais permet d'appliquer une largeur et une hauteur
  • none : n'apparait pas du tout
  • JSfiddle display et visibility

Mettre des couleurs

Les propriétés liées aux couleurs sont :

  • color (couleur du texte)
  • background-color (couleur du fond)

3 formats possibles :

  • color: blue; (voir liste des couleurs nommées)
  • color: rgb(0, 128, 255); (3 valeurs de rouge, vert et bleu de 0 à 255)
  • color: #0080FF; (idem, 3 valeurs RRGGBB en hexadécimal)

Le modèle en boîte

Box sizing

Par défaut, les propriétés width et height d'un bloc n'incluent que le contenu, pas le padding. Pour inclure le padding, utiliser box-sizing: border-box;

La pseudoclasse :hover

En ajoutant :hover à un sélecteur, on peut modifier le style de l'élément lorsque le pointeur de la souris le survole. Cette pseudoclasse peut être attribuée à n'importe quel balise.
Voir démonstration sur JSFiddle

Positionnement

Pour repositionner un bloc sur notre document, on donne à la propriété position la valeur :

  • relative : le bloc est décalé par rapport à sa position dans le flux
  • absolute : le bloc sort du flux et est positionné par rapport au bloc englobant
  • fixed : le bloc est fixé à une position sur la page

On définit ensuite top, right, left, bottom pour indiquer le décalage ou la position. Ex et + d'infos

Float : les éléments flottants

Un élément avec la propriété float sera positionné en premier par le navigateur, tout à gauche (float: left) ou à droite (float: right) de son contenant, puis le reste des éléments se positionnera autour.

Flexbox

  • CSS 3 (relativement bien supporté)
  • Introduit un nouveau mode d'affichage, les "boîtes flexibles", avec display: flex;
  • Enfin un moyen clair d'organiser ses éléments
  • Un jeu pour apprendre
  • Bac à sable pour tester
  • Guide complet

Flexbox - propriétés du contenant

  • flex-direction: définit l'axe principal (row, column, row-reverse, column-reverse)
  • justify-content: définit l'alignement des items selon l'axe principal (flex-start | flex-end | center | space-between | space-around)
  • align-items: définit alignement selon l'autre axe secondaire (flex-start | flex-end | center | stretch)
  • flex-wrap: par défaut, les items sont forcés sur une seule ligne (nowrap)

Flexbox - propriétés des items

  • align-self: permet de donner à un item spécifique un alignement différent de celui des autres (définit par align-items)(flex-start | flex-end | center | stretch)
  • flex-grow: indique la proportion d'espace restant que cet item va prendre. Par défaut, vaut 0. Si tous les items ont une valeur de 1, ils se partagent l'espace. Si un item a une valeur de 2, il va prendre deux fois plus d'espace que ceux à 1. Test
